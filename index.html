    Figma: My Go-To Choice

    Figma has become my primary design tool for several reasons:

    • Components: Advanced component system with variants
    • Plugins: Extensive plugin ecosystem
    • Web-based: Works on any platform without installation

    • Initial wireframing and user flows
    • High-fidelity mockups
    • Design system creation
    • Developer handoff

    Flutter Performance Tips I Learned

    Here's what actually improved the performance of my Flutter apps like Beekids Mobile and StockHive.

    Makram Ben Rhouma
    Read More

    Building real-world Flutter apps like Beekids Mobile and StockHive taught me valuable performance lessons. Here are the optimizations that made the biggest impact.

    1. Widget Optimization

    • Use const constructors: Prevents unnecessary rebuilds
    • ListView.builder: For large lists, always use builder pattern
    • RepaintBoundary: Isolate expensive widgets

    2. State Management

    Proper state management was crucial for Beekids Mobile:

    • Provider pattern: Used for global state
    • Local state: Kept widget-specific state local
    • Stream builders: For real-time chat features

    3. Image Optimization

    • Cached network images: Reduced loading times
    • Image compression: Optimized file sizes
    • Lazy loading: Load images only when needed

    4. Firebase Integration

    For real-time features in Beekids:

    • Firestore pagination: Load data in chunks
    • Offline persistence: Better user experience
    • Cloud Functions: Offload heavy processing

    These optimizations reduced app startup time by 40% and improved scroll performance significantly.

    Building a German-Style Developer Resume

    What I learned after testing my CV on ResumeMaker & ChatGPT to apply to jobs in Germany as a polyvalent profile.

    Makram Ben Rhouma
    Read More

    Creating a resume for the German job market requires understanding cultural expectations and technical requirements. Here's what I learned through research and AI assistance.

    German Resume Essentials

    • Professional Photo: Include a high-quality headshot
    • Personal Details: More detailed than US resumes
    • Chronological Order: Most recent experience first
    • Length: 2-3 pages is acceptable

    Technical Skills Section

    For developer roles, be specific about:

    • Programming Languages: Years of experience with each
    • Frameworks: Version numbers and project context
    • Tools & Technologies: Development environment preferences
    • Certifications: Include completion dates

    Project Descriptions

    German employers value detailed project information:

    • Technology Stack: Complete list of technologies used
    • Team Size: Your role in team projects
    • Results: Quantifiable achievements
    • Duration: Project timeline

    Language Skills

    Use the European framework:

    • German: B1 (learning towards B2)
    • English: C1 (professional working proficiency)
    • French: B2 (conversational)
    • Arabic: Native speaker

    Cover Letter Tips

    The German cover letter (Anschreiben) is crucial:

    • Formal Structure: Follow traditional business letter format
    • Company Research: Show knowledge of the company
    • Motivation: Explain why you want to work in Germany
    • Value Proposition: What you bring to the role

    AI Tools I Used

    • ChatGPT: Resume structure and content optimization
    • ResumeMaker: ATS-friendly formatting
    • Grammarly: Language polishing
    • LinkedIn: Networking and company research

    Key Takeaway: German employers value thoroughness and attention to detail. Invest time in creating a comprehensive, well-structured resume that tells your professional story clearly.
